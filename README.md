SAPER

This is game sapper in console.

First compile and run

  `make`

  `./saper.out`

Key guide:

q - quit

i - move up

k - move down

j - move left

l - move right

v - show visible field (number neighbour bombs, or bombs :-))

b - mark field as bomb

? - mark field as it can be bomb

ENJoy!
